## Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
git clone [repository-url]
cd tcn-comply-malta
npm install
```

### Development

```bash
npm run dev
# Open http://localhost:3000
```

### Build for Development Deployment

```bash
npm run build
```

### Deploy to Development Vercel

```bash
npm run deploy
```
